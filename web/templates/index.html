About This Application

This application helps you plan Mumbai local train routes while avoiding crowded stations. It uses:

  • Dijkstra's Algorithm - To find the shortest path between stations
  • Greedy Algorithm - To dynamically divert from crowded stations
  • Optimal Merge Pattern - To merge crowd data from multiple sources

The app covers Western, Central, and Harbor lines of Mumbai's suburban railway network.

Click the "Open Map View" button to start planning your journey.
